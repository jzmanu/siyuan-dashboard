# 功能介绍

siyuan-dashboard 作为数据统计插件，目前支持的功能如下：
- 支持天数、笔记、字数、标签数量统计。
- 支持使用天数百分比显示(实际使用天数/总天数)。
- 支持年、月、周总字数和平均字数统计。
- 支持年、月切换查看历史数据。
- 适配明亮、黑暗模式。

> 目前主要适配了桌面端，移动端暂时没有适配，如有需要后续会适配。
